Closeable - Assured Close

Some types of objects, like Sockets, need to be closed. This type of assurance is supported by classes which implement Closeable. The Reactor method addCloseable can then be used to ensure that the Closeable is closed when the Reactor is closed. But note that weak references are used to track closeables, and there is no assurance that close will be called on an object that has been garbage collected.
