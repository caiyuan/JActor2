Hello World! - A First Look

We will begin our exploration of JActor2 with a program, HelloWorld, that prints "Hello World!". In the process, we will encounter a number of classes which play a major role when using JActor. 

import org.agilewiki.jactor2.core.blades.misc.Printer;
import org.agilewiki.jactor2.core.plant.Plant;
import org.agilewiki.jactor2.core.messages.SyncRequest;

public class HelloWorld {
    public static void main(final String[] _args) throws Exception {
        Plant plant = new Plant();
        try {
            Printer.printlnAReq(plant, "Hello World!").call();
        } finally {
            plant.close();
        }
    }
}

Plant 

        Plant plant = new Plant();
        try {
            ...
        } finally {
            plant.close();
        }

A Plant is a subclass of Facility with a name of Plant. Every program must have one Plant. A program does not exit normally until its Plant is closed.

A facility provides a thread pool and other resources, with a default thread pool size of 20. Facilities are used by reactors, with every reactor having a reference to one facility. A program can have any number of facilities.

Printer 

            Printer.printlnAReq(plant, "Hello World!").call();

A printer is a BlockingBlade, which is to say an application object that uses a BlockingReactor. It prints to a PrintStream, System.out being the default.

The static method Printer.printlnAReq returns an AsyncRequest which, when called, prints the string. And if the facility has no printer, one is safely created.

The call method invokes the request to print the string. But it is printed from the printer's own thread, not from the main thread.

The call method of a request provides one way of invoking a request. The printRequest.call method sends the request to the blade's thread and blocks the main thread until a result is received. The call method is the only method in JActor which blocks a thread. (The call method can not be used by a blade, as it must be invoked from a non-facility thread.)
